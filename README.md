# 向僵尸开炮
前端现在这么厉害，我自己想了想，应该也能做游戏吧，就试着做了一下向僵尸开炮，但是这个游戏复杂程度远远高于我的想象，无奈，现在做了齐射 齐射 连发 穿透 增伤 2分裂 4分裂 冰子弹 火子弹 温压弹 冰暴发生器 干冰弹，加上技能升级的一些词条，太多了，多到爆炸，不一一列举了，我希望这个开源能帮到你们，但你们要是拿着乱搞，带上大手镯了，千万不要把为师供出来呀，开玩笑~



谁要是能拉到赞助，我们一起把这个游戏做大做强，使劲圈钱！！！！！哈哈哈！！！！！
